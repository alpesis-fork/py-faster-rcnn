Blob
==============================================================================

------------------------------------------------------------------------------
Data Structure
------------------------------------------------------------------------------


------------------------------------------------------------------------------
Operations
------------------------------------------------------------------------------

Operation types:

- ``reshape((rows, n, cols))``: reshape the dimensions of a blob 
- ``transpose(())``: transpose the cols of a blob

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
reshape()
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 

::

    blob:

        [[ -84,  -40,   99,   55],
         [-176,  -88,  191,  103],
         [-360, -184,  375,  199],
         [ -56,  -56,   71,   71],
         [-120, -120,  135,  135],
         [-248, -248,  263,  263],
         [ -36,  -80,   51,   95],
         [ -80, -168,   95,  183],
         [-168, -344,  183,  359]]


    blob.reshape((1, 9, 4)):

        [[[ -84,  -40,  99,  55],
         [-176,  -88,  191,  103],
         [-360, -184,  375,  199],
         [ -56,  -56,   71,   71],
         [-120, -120,  135,  135],
         [-248, -248,  263,  263],
         [ -36,  -80,   51,   95],
         [ -80, -168,   95,  183],
         [-168, -344,  183,  359]]]

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
transpose()
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 


::

    blob
    - shape: (1, 4 * n_anchors, height, width) -> (1, height, width, 4 * n_anchors)
    - blob.shape((0, 1, 2, 3)) -> blob.transpose((0, 2, 3, 1))
